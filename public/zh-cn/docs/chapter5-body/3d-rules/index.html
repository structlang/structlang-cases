      三维组合的规则

      三维组合的规则

      5.2 三维组合的规则

      体结构的组合规则比面结构更加复杂，但仍然遵循严格的逻辑。理解这些规则，是正确运用体结构的前提。

      一、三维判断的独立性原则

      体结构中的三个判断维度必须满足相对独立性：

      • 每个维度都有自己的判断对象和判断标准
      • 三个维度不能是同一判断的不同表达
      • 各维度之间允许有关联，但不能有因果决定关系

      正确示例：

      • X轴：我想不想跳槽？（意愿维度）
      • Y轴：我有没有能力跳槽？（能力维度）
      • Z轴：现在是不是好时机？（时机维度）

      错误示例：

      • X轴：我想不想跳槽？
      • Y轴：我希不希望换工作？（与X轴重复）
      • Z轴：我愿不愿意离职？（与X轴重复）

      二、结构张力的立体计算

      在面结构中，我们用二维距离计算结构压强：

      SP = √[(Δx)² + (Δy)²]

      在体结构中，张力计算扩展为三维：

      SP = √[(Δx)² + (Δy)² + (Δz)²]

      其中，理想状态通常是（+1, +1, +1）或（-1, -1, -1），即所有维度保持一致。

      张力分级：

      • 低张力（SP ≤ 1.0）：最多一个维度不一致
      • 中张力（1.0 < SP ≤ 2.0）：两个维度不一致
      • 高张力（SP > 2.0）：三个维度互相冲突

      三、立体象限的状态含义

      8个基本象限各自代表不同的认知与行为状态：

      一致型象限（低张力）

      编码 状态特征 心理体验 典型表现
      +++ 三维全通 顺畅、自信、积极 想要→能够→时机对→行动
      — 三维全断 平静、冷漠、回避 不想→不能→不合适→放弃

      部分冲突型象限（中张力）

      编码 状态特征 心理体验 典型表现
      ++- 意愿能力充足，时机不对 焦急、等待 “想做也能做，但现在不行”
      +-+ 有意愿有时机，能力不足 焦虑、学习冲动 “想做时机也对，但还不够格”
      -++ 不想做但有能力且时机对 纠结、被推动 “别人都觉得我应该做”
      +– 只有意愿，其他都不具备 无力、幻想 “想法很美好，现实很骨感”
      -+- 只有能力，意愿时机都不对 浪费、遗憾 “明明可以，但就是不想做”
      –+ 只有时机，意愿能力都缺乏 错失、茫然 “机会来了但抓不住”

      全冲突型象限（高张力）

      当三个维度两两冲突时，会产生极高的结构张力，通常难以长期维持，必然向其他象限跳变。

      四、跳变规律与路径预测

      体结构中的跳变比面结构更加复杂，但仍有规律可循：

      4.1 最小跳变原则

      个体倾向于进行单维度跳变（一次只改变一个判断维度），而不是同时改变多个维度。

      例如：从（+, -, +）最可能跳变到：

      • （-, -, +）：改变意愿维度
      • （+, +, +）：改变能力维度
      • （+, -, -）：改变时机维度

      4.2 张力释放倾向

      高张力状态会优先向低张力状态跳变：

      • 全冲突型 → 部分冲突型 → 一致型

      4.3 外部触发因子

      跳变往往需要外部触发：

      • 信息输入：新的认知改变某个维度的判断
      • 情绪波动：情绪状态影响意愿维度
      • 环境变化：外部条件改变能力或时机维度
      • 社会压力：他人期望影响判断标准

      五、体结构的建模步骤

      要正确运用体结构，需要遵循以下建模步骤：

      1. 确认复杂性：判断是否真的需要三个维度
      2. 选择维度：确定三个相对独立的判断维度
      3. 定位象限：判断当前状态落在哪个象限
      4. 计算张力：评估当前状态的稳定性
      5. 预测跳变：分析可能的状态迁移方向
      6. 设计干预：如果需要，设计促进或阻止跳变的策略

      六、组合规则的约束条件

      并非所有的三个判断维度都能构成有效的体结构：

      有效条件：

      • 三个维度都与同一决策对象相关
      • 各维度相对独立，不存在严格的因果关系
      • 每个维度都可以独立产生正负判断
      • 三个维度的组合能够覆盖该决策的主要考虑因素

      无效情况：

      • 某个维度是另一个维度的逻辑后果
      • 三个维度实际指向不同的决策对象
      • 某个维度缺乏主观判断性（纯客观约束）
      • 维度设置过于抽象，无法对应具体判断
      ℹ️
      实践建议：在建立体结构时，先用面结构尝试建模，只有当面结构明显无法容纳所有重要判断维度时，才引入第三维度。

      体结构的组合规则为我们提供了处理复杂决策的系统化方法，但关键在于正确识别和应用这些规则，而不是为了复杂而复杂。
